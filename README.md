# teacher-mbti
班主任 H-MBTI 趣味测评

这是一个仿照 MBTI 形式制作的班主任风格趣味测评网页。

它用四个维度概括班主任的带班风格：

- E / I：推动型 vs 引导型
- S / N：基础型 vs 成长型
- T / F：规则型 vs 关系型
- J / P：计划型 vs 弹性型

本项目仅供自娱自乐和交流，请注意这不是专业心理测评。结果不代表谁更优秀，也不用于评价班主任好坏。它更像一次班级里的小观察。关于这一想法的创意来自于和我的舍友lsw老师的一次闲聊，后续的工作则来自于gpt的努力（没错在这些过程当中本人都是一个到处闲聊的角色，如果大家喜欢可以转发给朋友玩一玩，如果不喜欢也请不要骂我）
